# TaskFlow - Task Manager Landing Page

TaskFlow is a modern, premium landing page concept for a fictional task management product. It is designed to look and feel like a real startup website, with polished UI, smooth interactions, and a responsive layout.

## Overview

This project showcases a high-end marketing page built with plain HTML, CSS, and JavaScript. It includes:
- A split hero section with product messaging and CTA buttons
- A rich Kanban-style dashboard preview
- Feature highlight cards
- A detailed product preview section
- A strong call-to-action section
- A clean, minimal footer

## Features

- Premium dark UI with gradient accents
- Glassmorphism cards and layered depth effects
- Smooth reveal animations on scroll
- Animated progress bars and counters
- Subtle 3D tilt interaction for UI panels
- Fully responsive design for mobile, tablet, and desktop

## Tech Stack

- HTML5
- CSS3 (Flexbox + Grid, custom properties, responsive breakpoints)
- JavaScript (Intersection Observer, lightweight UI animations)

Task Manager/

## Getting Started

1. Clone or download this project.
2. Open the project folder.
3. Open `index.html` in your browser.

Optional local server (recommended):

- VS Code Live Server extension, or
- Run a simple static server from the folder (for example with Python):

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Customization

- Update text content in `index.html`
- Tweak colors, shadows, and spacing in `style.css`
- Adjust animation behavior in `script.js`

## Purpose

The goal of this project is to demonstrate modern frontend UI skills and create a realistic SaaS-style landing page suitable for portfolio presentation or design inspiration.
