# TaskFlow – Task Manager Landing Page

A modern, high-fidelity landing page for a fictional task manager designed to feel like a real startup product.

## 🔗 Live Demo

👉 https://taskflow-by-minkush.netlify.app/

## 📦 GitHub Repository

👉 https://github.com/minkushgoyal/taskflow-landing-page

---

## ✨ Key Highlights

* Realistic **Kanban-style dashboard UI**
* Modern **glassmorphism and layered design**
* Smooth **scroll reveal animations**
* Interactive elements (counters, progress bars, tilt effect)
* Clean **responsive layout** for mobile and desktop

---

## 🛠 Tech Stack

* HTML5
* CSS3 (Flexbox, Grid, custom properties)
* JavaScript (Intersection Observer, animations)
